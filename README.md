# algo-helper-be
Backend Service for Algo-Helper


## Start up Instructions
> **"npm start"** to start the server

## User Endpoints:

### POST: 
> **/register**   
- register user

### DELETE: 
> **/delete/user**     
- delete user   

### GET: 
> **/signin**   
- gets user information

## Card Endpoints:

### POST: 
> **/create**    
- create a card
### DELETE: 
> **/delete**    
- delete a card
### GET: 
> **/cards**     
- gets cards

### UPDATE: 
> **/update/field**    
- update card field     
> **/update/position**   
- update card position
